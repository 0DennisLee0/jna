    public abstract class IntegerType
extends java.lang.Number
implements NativeMapped
    Represents a native integer value, which may have a platform-specific size (e.g. long on unix-based platforms). May optionally indicate an unsigned attribute, such that when a value is extracted into a larger-sized container (e.g. int retrieved via Number.longValue(), the value will be unsigned. Default behavior is signed.
